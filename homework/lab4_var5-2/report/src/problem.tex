\CWHeader{Лабораторная работа \textnumero 4}

\CWProblem{
Необходимо реализовать один из стандартных алгоритмов поиска образцов для указанного алфавита.

{\bfseries Вариант алгоритма:} Поиск большого количества образцов при помощи алгоритма Ахо--Корасик.

{\bfseries Вариант алфавита:} { Числа в диапазоне от 0 до $2^{32} - 1$. }

Искомые образцы задаются по одному на строку до первой пустой строки. После неё следует текст,
состоящий из слов или чисел. Для каждого найденного вхождения требуется вывести номер строки текста,
номер слова в строке, с которого начинается вхождение, и порядковый номер образца. Нумерация строк,
слов и образцов начинается с единицы.
}
