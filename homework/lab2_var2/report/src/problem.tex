\CWHeader{Лабораторная работа \textnumero 2}

\CWProblem{
Требуется разработать программу-словарь на основе заданной структуры данных.

{\bfseries Вариант структуры данных:} Красно-чёрное дерево.

{\bfseries Вариант ключа:} { \normalfont Регистронезависимая последовательность букв английского алфавита длиной не более 256 символов. }

{\bfseries Вариант значения:} { \normalfont\ttfamily Целое беззнаковое число от 0 до $2^{64} - 1$. }

Словарь должен поддерживать операции вставки, удаления, поиска, сохранения в бинарный файл и загрузки из бинарного файла. При загрузке корректного файла новый словарь должен заменять текущий; при ошибке чтения или несовпадении формата рабочий словарь должен оставаться без изменений. Пустой или отсутствующий файл при загрузке считается корректным и соответствует пустому словарю.
}
